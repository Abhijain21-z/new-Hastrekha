## Browser Rendering Path

```
1. Load HTML
   ↓
2. Parse CSS (animations defined)
   ↓
3. Load JS (Three.js, React Three Fiber)
   ↓
4. Initialize WebGL Context
   ↓
5. Create 3D Scene
   ├─ Add lights
   ├─ Add planets
   ├─ Add stars
   └─ Add materials
   ↓
6. Start Animation Loop (requestAnimationFrame)
   ├─ Update orbital positions
   ├─ Update rotations
   ├─ Render frame
   └─ Repeat 60x per second
   ↓
7. Scroll Listener Active
   ├─ Calculate scroll position
   ├─ Update CSS animations
   ├─ Trigger reveals
   └─ Parallax updates
```

## Customization Visual Examples

### Change Earth to Purple
```typescript
// Before:
color: '#4A90E2',    // Blue
emissive: '#2E5C8A'  // Darker blue

// After:
color: '#9D4EDD',    // Purple
emissive: '#6A0572'  // Dark purple
```

### Slow Down Saturn
```typescript
// Before:
speed: 0.002  // Current slow speed

// After:
speed: 0.0005  // Even slower!
```

### Make Stars More Visible
```typescript
// In SimpleStars()
<pointsMaterial 
  size={0.5}              // Larger stars
  opacity={0.9}           // More visible (was 0.7)
/>
```
